i. Inspiracje (słowa, skojarzenia, pomysły)
- notatki, porządek, szybkie odnajdywanie
- tagi, katalogowanie, foldery, kolekcje
- wyszukiwarka kontekstowa, filtrowanie zaawansowane
- offline-first, synchronizacja, konflikt merge
- import (PDF, Markdown, obraz), OCR podstawowy
- flashcards, podsumowania automatyczne, highlighty
- skróty klawiszowe, tryb ciemny, responsywność
- pinowanie, ulubione, przypomnienia przed egzaminem
- metadata: wykład, przedmiot, prowadzący, data
- UX: minimalizm, szybkie dodawanie, niskie tarcie

ii. Krótka lista elementów projektu
- interfejs użytkownika (web/mobile PWA)
- lokalna baza danych (IndexedDB / SQLite)
- API synchronizacji (opcjonalne)
- silnik wyszukiwarki full-text
- system tagów i kategorii
- import/eksport (Markdown, PDF, JSON)
- tryb offline i konfliktów
- moduł OCR (opcjonalnie)
- backend autoryzacji (opcjonalny)
- testy jednostkowe i e2e
